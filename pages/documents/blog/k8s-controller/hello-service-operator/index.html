Introduction

This document outlines the implementation plans for the Hello Service Operator, a Kubernetes operator that manages Hello Service resources.

Prerequisites

  • Go latest

  • Kube builder

Install or upgrade to latest Go

$ brew update
$ brew upgrade go

current version:

Before upgrade
$ go version

go version go1.24.5 darwin/arm64
After upgrade
$ go version
go version go1.25.6 darwin/arm64

Install latest Kube builder

$ brew update
$ brew install kubebuilder

current version:

After Installation
$ kubebuilder version

KubeBuilder:          v4.11.1
Kubernetes:           1.35.0
Git Commit:           4fbe8f6c63c6eb9a2a5cc440acbedad6b60d3225
Build Date:           2026-01-27T16:16:09Z

SSO Foundry

Single Sign-on for Service Foundry

SSO Foundry is a comprehensive solution for implementing Single Sign-On (SSO) in your Service Foundry environment. It provides a seamless authentication experience across all Service Foundry modules, ensuring secure and efficient user access management.

Key Features

Seamless Integration

Integrates with existing Service Foundry modules to provide a unified authentication experience.

OpenID Connect Support

Supports OpenID Connect for secure and standardized authentication.

User Management

Provides tools for managing users, roles, and permissions across the Service Foundry ecosystem. url: sso-foundry/

Open Source Foundation

Built on open source technologies, ensuring transparency and community support. The SSO Foundry leverages Keycloak for identity management, providing a robust and flexible solution for user authentication and authorization. OAuth2 Proxy is used to secure access to applications and services, enabling fine-grained control over user permissions and access rights.

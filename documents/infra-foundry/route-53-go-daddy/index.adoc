= Go Daddy Route 53 DNS Setup

== Overview

Route 53

== Create Hosted Zone in Route 53

1. Log in to your AWS Management Console.
2. Navigate to the Route 53 service.
3. Click on "Hosted zones" in the left sidebar.
4. Click on the "Create hosted zone" button.
5. Enter your domain name (e.g., `servicefoundry.ca`) in the "Domain name" field.
6. Optionally, add a comment and select the type as "Public hosted zone".
7. Click on the "Create hosted zone" button to create the hosted zone.

You will see a list of DNS records, including NS (Name Server) and SOA (Start of Authority) records.

----
ns-1371.awsdns-43.org.
ns-835.awsdns-40.net.
ns-1601.awsdns-08.co.uk.
ns-458.awsdns-57.com.
----

== Update Name Servers in Go Daddy

1. Log in to your Go Daddy account.
2. Navigate to the "My Products" page.
3. Find your domain (e.g., `servicefoundry.ca`) and click on the
    "DNS" button next to it.
4. In the "Nameservers" section, click on the "Change" button.
5. Select "Custom" from the options.

.Go Daddy Nameserver Settings
----
ns11.domaincontrol.com
ns12.domaincontrol.com
----

6. Replace the existing nameservers with the four nameservers provided by Route 53.

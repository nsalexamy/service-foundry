= How to create a Kubernetes Operator - Hello Service Operator

== Introduction

This document outlines the implementation plans for the Hello Service Operator, a Kubernetes operator that manages Hello Service resources. 

== Prerequisites

- Go latest 
- Kube builder

=== Install or upgrade to latest Go

[source,shell]
-----
$ brew update
$ brew upgrade go
-----

current version:

.Before upgrade
[source,shell]
-----
$ go version

go version go1.24.5 darwin/arm64
-----

.After upgrade
[source,shell]
-----
$ go version
go version go1.25.6 darwin/arm64
-----

=== Install latest Kube builder

[source,shell]
-----
$ brew update
$ brew install kubebuilder
-----

current version:

.After Installation
[source,shell]
-----
$ kubebuilder version

KubeBuilder:          v4.11.1
Kubernetes:           1.35.0

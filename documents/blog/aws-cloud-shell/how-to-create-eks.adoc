== Environmental variables


[source,shell]
----
$ aws configure list
----

Verify that the following environment variables are set in your AWS Cloud Shell:

[source,shell]
----
$ env | grep -e AWS -e EKS
----

Required environment variables:

* AWS_REGION - Already set in Cloud Shell, e.g., ca-central-1.
* AWS_ACCOUNT_ID - Your AWS account ID.
* EKS_CLUSTER_NAME - Name of your EKS cluster, e.g., my-eks
* EKS_DESIRED_NODES - Desired number of nodes, e.g., 2
* EKS_MIN_NODES - Minimum number of nodes, e.g., 1
* EKS_MAX_NODES - Maximum number of nodes, e.g., 3

$EKS_MIN_NODES <= $EKS_DESIRED_NODES <= $EKS_MAX_NODES

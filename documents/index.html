Technical Documents

Technical documentation for the Service Foundry, including architecture, getting started guides, and detailed information on each foundry.

🔐 SSO Foundry

Centralize identity and access using Keycloak, OAuth2 Proxy, and Traefik for secure single sign-on.

📊 Observability Foundry

Integrated monitoring, logging, and tracing using OpenTelemetry, Prometheus, and Grafana.

🛠 Infra Foundry

Provision and manage core infrastructure like ingress controllers, certificates, and core services with Helm.

🧩 Backend Foundry

A Spring-based microservice runtime with OpenAPI, Keycloak integration, and API Gateway configuration.

📦 Big Data Foundry

A scalable data platform with Apache Spark, Kafka, Airflow, and dbt for processing EMR and analytics pipelines.

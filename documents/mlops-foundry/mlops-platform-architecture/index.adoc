== Introduction

MLOps Platform is a platform for data scientists and machine learning engineers to build, train, and deploy machine learning models.

MLOps Platform is built on top of Service Foundry and provides a set of tools and services to help data scientists and machine learning engineers to build, train, and deploy machine learning models.
